**Cursor**

요청하신 `.cursorrules` 규칙 추가로 이해했습니다. 먼저 Cursor 규칙 작성 지침을 확인한 뒤, 현재 파일 구조에 맞춰 붙여넣기 가능한 문구를 반영하겠습니다.

현재 요청은 기존 `.cursorrules` 파일에 “항상 적용되는 절대 규칙”을 추가하는 범위가 명확하므로, 파일 내용을 확인한 뒤 기존 문체에 맞춰 최소 변경하겠습니다.

`.cursorrules`는 한국어 규칙 목록 형태로 정리되어 있습니다. 기존 형식을 유지해 `절대 규칙` 섹션을 추가하겠습니다.

`.cursorrules`에 `절대 규칙` 섹션을 추가했습니다.

추가된 내용은 파일 입력 스트림에서 `weight`가 누락되어 `0`으로 읽히는 경우, 같은 나이대 예: `20~29` 표본의 평균 몸무게로 반드시 치환하도록 명시합니다.
