# SHEALTH Cursor AI 절대 규칙 추가 보고서

## 1. 작성 목적

본 문서는 SHEALTH C++17 프로젝트의 `.cursorrules` 파일에 추가한 파일 입력 스트림의 누락 몸무게 처리 규칙을 정리한 보고서입니다.

해당 규칙은 Cursor AI가 SHEALTH 프로젝트에서 QA, 리팩토링, 테스트 작성, 코드 수정 작업을 수행할 때 항상 따라야 하는 절대 규칙으로 정의되었습니다.

## 2. 변경 대상

- 대상 파일: `.cursorrules`
- 변경 섹션: `절대 규칙`
- 적용 범위: 파일 입력 스트림에서 읽은 건강 데이터의 `weight` 컬럼 처리

## 3. 추가된 절대 규칙

파일 입력 스트림의 컬럼 항목 중 몸무게(`weight`)가 누락되어 `0`의 값으로 읽히는 경우, 해당 데이터와 같은 나이대의 평균 몸무게로 반드시 치환하도록 규칙을 추가했습니다.

최종 반영된 규칙은 다음과 같습니다.

```text
절대 규칙:
- 파일 입력 스트림의 컬럼 항목 중 몸무게(weight)가 누락되어 0의 값으로 읽히는 경우, 해당 데이터와 같은 나이대의 평균 몸무게로 반드시 치환합니다. 예를 들어 Age가 27이고 몸무게가 0이면 수집된 Age 20~29 표본의 평균 몸무게로 치환하고, Age가 37이고 몸무게가 0이면 수집된 Age 30~39 표본의 평균 몸무게로 치환합니다. 40대, 50대, 그 이후 나이대도 동일한 10년 단위 나이대 표본의 평균 몸무게를 적용합니다.
```

## 4. 적용 기준

몸무게가 `0`으로 읽힌 데이터는 실제 몸무게가 0이라는 의미가 아니라, 파일 입력 스트림에서 몸무게 값이 누락된 것으로 간주합니다.

누락 몸무게는 같은 10년 단위 나이대 표본의 평균 몸무게로 보정합니다.

- Age 27, weight 0: Age 20~29 표본의 평균 몸무게로 치환
- Age 37, weight 0: Age 30~39 표본의 평균 몸무게로 치환
- Age 40대, 50대, 이후 나이대: 각 10년 단위 나이대 표본의 평균 몸무게로 치환

## 5. 기대 효과

이 규칙을 통해 Cursor AI는 SHEALTH 프로젝트의 파일 입력 처리 로직을 수정하거나 테스트할 때 누락 몸무게 보정 정책을 일관되게 적용할 수 있습니다.

특히 BMI 계산, 나이대별 통계, 회귀 테스트 작성 과정에서 `weight == 0` 데이터를 잘못된 정상 입력으로 처리하지 않고, 같은 나이대의 대표값으로 보정해야 한다는 도메인 규칙을 명확히 따르게 됩니다.
